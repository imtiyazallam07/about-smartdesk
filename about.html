Version History

Track the evolution of SmartDesk through every update.

v3.3.1 LATEST
  • New Data backup: Export your data to a local file and import it anytime for backup and restore.
  • Fix Fixed notifications not triggering correctly for recurring tasks.
v3.2.0
  • New Home screen widget: Quick attendance overview support.
  • Improve Improved attendance analytics and minor UI enhancements.
  • Fix Fixed auto-status switching based on subject checkboxes.
v3.0.0
  • Major Complete UI/UX overhaul for speed and intuition.
  • New Total & Monthly Attendance: View percentages overall or by month.
  • New Subject Analytics: Visual graphs for session trends.
  • New Swipe Gestures: Mark attendance directly from home (Right: Present, Left: Absent).
v2.2.0
  • New Cloud Timetable: Seamlessly fetch and download your latest timetable from the cloud.
  • Improve UI components updated for cloud data handling.
  • Fix Stability improvements and minor glitch fixes.
v2.1.0
  • New Onboarding screen for a smooth first-time experience.
  • Improve Smarter Updates: Auto-check every 24h, Manual check every 1h.
  • UI Dynamic Ring Colors: 80%+ (Green), 75-79% (Amber), 60-74% (Orange), <60% (Red).
v2.0.0
  • New QR Timetables: Import/Export schedules using QR codes.
  • New Branch Selection: curriculum now displays based on your branch/year.
  • Change Attendance view changed from donut chart to listed card layout.
  • Fix Fixed notifications triggering after attendance was already marked.
v1.9.0
  • New Central Dashboard: Combined view for Attendance, Events, Classes, and Library.
  • New Visual Tracker: Intuitive way to monitor presence/absence.
  • New Library Reminders: Automated notifications for return dates.
  • New Full Dark Mode: Respects system theme preferences.
